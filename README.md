# Calculator Android Assignment

This project is a simple Android Calculator application developed using Android Studio. The application performs basic arithmetic operations and demonstrates fundamental Android development concepts such as activities, layouts, and user interface design.

## Features
- Basic arithmetic operations (Addition, Subtraction, Multiplication, Division)
- Simple and user-friendly interface
- Splash screen implementation
- Developed using Java and XML

## Technologies Used
- Java
- Android Studio
- XML Layouts
- Android SDK

## Project Structure
- MainActivity.java – Handles the main calculator functionality
- SplashScreen.java – Displays the splash screen when the app starts
- activity_main.xml – Layout for the main calculator interface
- activity_splash_screen.xml – Layout for the splash screen

## How to Run the Project
1. Download or clone the repository.
2. Open the project in Android Studio.
3. Let Gradle sync the project dependencies.
4. Run the application on an Android emulator or physical device.
